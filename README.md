# 📌 Overview

SMART HR 2.0 is a complete Human Resource Management (HRM) solution designed to automate and simplify day-to-day HR operations.

Instead of relying on expensive biometric attendance devices, the application leverages **GPS location verification**, **QR technology**, and **OTP authentication** to securely manage employee attendance from anywhere.

Built entirely with **Flutter**, the application runs seamlessly across Android, iOS, Windows, macOS, Linux, and the Web from a single codebase.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Pranav-Gor/SMART-HR-2.0.git
```

```
cd SMART-HR-2.0
```

Install Packages

```bash
flutter pub get
```

Run Application

```bash
flutter run
```

---

# 🏗️ Build

Android

```bash
flutter build apk --release
```

iOS

```bash
flutter build ios --release
```

Web

```bash
flutter build web
```

Windows

